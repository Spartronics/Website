The galleries have their pictures, thumbnails, descriptions and titles defined in a .JSON file.
Copy one from /images somewhere and edit it; I'm not going to write it out here.
Unlike everything else on the site, paths here are relative to the .JSON, not to the website root.

JSON attributes

  • "title": The title that appears above the image. Can be left out, but if you do, try to keep the same among all the pictures.
  • "desc": The text that appears below the image. Keep short. Entirely optional.
  • "thumb": Thumbnail shown on the select bar. Mandatory unless autoscrolling.
  • "pic": The picture to display. Mandatory.
